📋 Instrucciones de Uso

Cómo usar los instrumentos:

1. Abrir: Haz clic en "Abrir" para lanzar el instrumento en una ventana nueva

2. Rellenar: Completa los datos necesarios en los campos disponibles

3. Editar (opcional): Personaliza títulos, encabezados o escalas si es necesario

4. Guardar: Descarga los datos en JSON para conservar el registro

5. Limpiar: Elimina todos los datos para una nueva sesión independiente

6. Cerrar: Cierra el instrumento (se preguntará si deseas guardar)


Nota importante: Los instrumentos son completamente independientes. Cada vez que los abres, inician vacíos sin recordar usos anteriores. Esto asegura que los datos mostrados siempre sean nuevos y no contaminados.
